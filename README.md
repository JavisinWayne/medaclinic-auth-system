# 🏥 Medaclinic Login System

Sistema completo de autenticación médica con Backend + Frontend, gestión de citas y dashboard administrativo.

---

## ✨ Características

- ✅ Backend Node.js + Express + MySQL
- ✅ Autenticación JWT segura
- ✅ Registro / Login con bcrypt
- ✅ Dashboard por roles (admin / usuario)
- ✅ CRUD de citas médicas
- ✅ Exportación a Excel
- ✅ Sistema de alertas en frontend
- ✅ Interfaz responsive
- ✅ Datos demo incluidos
